  Advent of Code 2025 (C++)

━━━━━━━━━━━━━━━━━━━━━━
  Requirements
━━━━━━━━━━━━━━━━━━━━━━
- CMake
- C++ compiler (GCC or Clang)
- Build system (any one of these):
  - make
  - ninja
  - or any CMake-supported generator

━━━━━━━━━━━━━━━━━━━━━━
  Build
━━━━━━━━━━━━━━━━━━━━━━

```sh
mkdir build;
cd build;
cmake ..;
cmake --build .
```

━━━━━━━━━━━━━━━━━━━━━━
  Run
━━━━━━━━━━━━━━━━━━━━━━

```sh
./run_day <day> <part>
```

Example:
```sh
./run_day 1 2
```
